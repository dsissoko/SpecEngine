## Interfaces de données
Les interfaces entre données (échanges de structures, vues de lecture, événements de domaine, etc.)
sont considérées comme naturellement visibles dans les diagrammes (MCD, classes…).

Aucun catalogue spécifique n’est maintenu ici : les contrats de données exposés
sont référencés depuis les chapitres d’interfaces fonctionnelles, logicielles ou techniques
lorsque cela est nécessaire.
